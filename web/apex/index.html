Pollen

A local-first, zero-trust, leaderless, emergent mesh and WASM runtime, in a single static binary.

curl -fsSL https://pln.sh/install.sh | bash
  • Ergonomic. Opinionated defaults, opt-in configuration.
  • Self-organising. No scheduler, no leader. Topology, placement and routing emerge from local state.
  • Partition-tolerant. Both sides keep running; state converges on rejoin.
  • QUIC transport. One multiplexed, encrypted link per peer carries gossip, services and seeds.
  • Cryptographic admission. No shared secrets, no firewall rules — every link is mTLS.
